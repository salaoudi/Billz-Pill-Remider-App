# Billz-Pill-Remider-App
 Bill lets users schedule a time to when they'll take thier next pill. 
 
** Features:**
 - ADD AND SCHEDULE A PILL
 - ADD AN ICON AND AN ICON COLOR
 - UPDATE PILLS
 - PIN PILLS TO TOP
