# Billz-Pill-Remider-App
 Billz lets users schedule a time to when they'll take thier next pill. 
 
 
 
**TECH USED**

- SWIFT | 
- SWIFTUI | 
- USERDEFAULTS | 
- COREDATA 
- LOCAL NOTIFICATIONS |

** FEATURES**
 - ADD AND SCHEDULE A PILL
 - ADD AN ICON AND AN ICON COLOR
 - UPDATE PILLS
 - PIN PILLS TO TOP
